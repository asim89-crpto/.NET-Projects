# SentimentAnalysisApp

ASP.NET Core Web API that performs sentiment analysis using ML.NET.

## Features
- REST endpoints for submitting text
- ML.NET sentiment classification (Positive / Negative / Neutral)
- SQL Server (Entity Framework Core)
- Ready for deployment on AWS (EC2 / RDS / Elastic Beanstalk)

## How to Run
1. Open `SentimentAnalysisApp.sln` in Visual Studio
2. Update `appsettings.json`
3. Press F5 to run the API
